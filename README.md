# 🏦 Bank Customer Churn Analysis Dashboard

## 📌 Project Overview
This project presents an interactive Power BI dashboard designed to analyze customer churn patterns in the banking sector. The dashboard helps identify the major factors influencing customer attrition using demographic, financial, and behavioral data insights.

---

## 🎯 Objective
The primary goal of this project is to:

- Analyze customer churn trends
- Identify high-risk customer segments
- Understand the impact of age, balance, credit score, gender, and products on churn
- Support data-driven business decisions for customer retention strategies

---

## 📊 Dashboard Features

### KPI Metrics
- Total Customers
- Churned Customers
- Active Customers
- Average Age
- Average Balance
- Average Credit Score
- Average Salary
- Average Tenure

### Visual Insights
- Customer churn distribution
- Churn analysis by gender
- Age group most likely to exit
- Balance impact on customer churn
- Product usage vs churn rate
- Country-wise customer analysis

### Interactive Filters
- Gender
- Country

---

## 🛠️ Tools & Technologies Used

| Tool | Purpose |
|------|----------|
| Power BI | Data Visualization & Dashboard Creation |
| Excel / CSV | Data Source |
| DAX | Calculated Measures & KPIs |
| Power Query | Data Cleaning & Transformation |

---

## 📈 Business Insights
- Customers with lower engagement are more likely to churn.
- Middle-aged customers show higher exit probability.
- Balance and number of products significantly impact retention.
- Female customers showed slightly higher churn rates in the dataset.
- Country-wise variations indicate different customer behaviors.
